Option 1: Iframe Integration (Recommended)

✓ Advantages

  • Drop-in integration - just add one HTML tag
  • No JavaScript conflicts with your existing site
  • Automatic updates from our platform
  • Zero maintenance on your end
  • Consistent styling and functionality

✗ Limitations

  • Fixed size iframe boundaries
  • Limited customization of internal elements

Implementation:

<iframe src="http://localhost:5000/iframe_widget.html" width="100%" height="700" frameborder="0" style="border-radius: 10px;"> </iframe>

Option 2: JavaScript Widget Integration

✓ Advantages

  • Full customization control
  • Better user experience (no iframe boundaries)
  • Can integrate with your existing user system
  • Dynamic sizing and responsive design

✗ Drawbacks

  • Requires JavaScript integration
  • Potential styling conflicts
  • Need to handle API calls from your frontend
  • More complex user balance management

Implementation:

<div id="mystery-boxes"></div> <script> const apiConfig = { endpoint: 'http://localhost:5000', apiKey: 'test' }; async function loadMysteryBoxes() { const response = await fetch(`${apiConfig.endpoint}/api/v1/boxes`, { headers: { 'X-API-Key': apiConfig.apiKey } }); const data = await response.json(); // Render boxes in your custom styling document.getElementById('mystery-boxes').innerHTML = data.data.map(box => ` <div class="mystery-box" onclick="openBox(${box.id})"> <h3>${box.name}</h3> <p>$${box.price}</p> </div> `).join(''); } async function openBox(boxId) { const response = await fetch(`${apiConfig.endpoint}/api/v1/open-box`, { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': apiConfig.apiKey }, body: JSON.stringify({ boxId: boxId, userId: 'your_user_id', userBalance: 1000 }) }); const result = await response.json(); alert(`You won: ${result.data.item.name}!`); } loadMysteryBoxes(); </script>
